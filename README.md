# Event Posters Web App

A simple frontend project that displays a list of upcoming events in a poster‑style format. Each event includes its name, description, contact info, and optional booking link.

## Features
- Shows sample events on a clean, card‑like UI
- Easy to customize with your own data
- Built with HTML
- Hosted on Netlify

## Live Demo
https://69ab49198dc1ec9c6110950c--event-posters.netlify.app/

## How to Use
1. Clone the repo
2. Open `index.html` in a browser (or run a dev server)
3. Update the event data as needed
